metodologia de BEN

ESTA ES LA METODOLOGIA DE BEN

-se puede mandar a llamar el primer hijo de el padre div con un .form__input:first-child
-se puede poner esto .form__input--active o --algo que indentifique el input o lo que sea que queramos en especifico para modificarlo, asi este tendra la especificidad correcta.
-los __ son para bloque como divs y los -- son para cosas ya mucho mas especificas
-cabe resaltar que esta es solo una metodologia los nombres de las clases pueden ser cualquiera pero es recomendable seguir sierto patron para que nuestro codigo sea mas propenso a tener menos errores
